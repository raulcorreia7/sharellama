# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) for ShareLlama.

## What is an ADR?

An ADR captures a significant architectural decision along with its context and consequences.

## Structure

```
NNNN-title-with-dashes.md
```

- **NNNN**: Sequential number (0001, 0002, ...)
- **title**: Short description of the decision

## Template

```markdown
# NNNN - Title

## Status

Proposed | Accepted | Deprecated | Superseded

## Context

What is the issue, forces, and background?

## Decision

What is the change or action being taken?

## Consequences

What are the results, trade-offs, and impact?

## Alternatives Considered

What else was considered and why was it rejected?
```
